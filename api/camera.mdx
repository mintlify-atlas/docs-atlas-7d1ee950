## Class signature

## Static factory

### `Camera.create(camera_id, model, focal_length, width, height)`

Create a camera with default parameters for the given model.

<ParamField path="camera_id" type="int" required>
  ID to assign to the new camera.
</ParamField>

<ParamField path="model" type="pycolmap.CameraModelId or str or int" required>
  Camera model.
</ParamField>

<ParamField path="focal_length" type="float" required>
  Focal length in pixels.
</ParamField>

<ParamField path="width" type="int" required>
  Image width in pixels.
</ParamField>

<ParamField path="height" type="int" required>
  Image height in pixels.
</ParamField>

```python

```

## Intrinsic utilities

### `mean_focal_length()`

Return the mean of `focal_length_x` and `focal_length_y`.

### `calibration_matrix()`

Return the 3×3 calibration matrix **K** computed from the current parameters.

### `focal_length_idxs()`

Return the indices of focal length parameters within `params`.

### `principal_point_idxs()`

Return the indices of principal point parameters within `params`.

### `extra_params_idxs()`

Return the indices of extra (distortion) parameters within `params`.

## Projection methods

### `img_from_cam(point2d)`

Project a point from the camera coordinate frame (normalized image plane) to pixel coordinates. Also accepts a batch as an Nx2 array or a list of `Point2D` objects.

<ParamField path="point2d" type="numpy.ndarray (2,) or (N,2) or list[Point2D]" required>
  Input point(s) in camera space.
</ParamField>

### `cam_from_img(point2d)`

Back-project a pixel coordinate to the normalized camera plane (undistorts and divides by focal length). Also accepts Nx2 arrays and lists of `Point2D`.

<ParamField path="point2d" type="numpy.ndarray (2,) or (N,2) or list[Point2D]" required>
  Input pixel coordinate(s).
</ParamField>

### `cam_from_img_threshold(threshold)`

Convert a pixel-space threshold to the equivalent threshold in normalized camera space.

### `rescale(width, height)` / `rescale(scale)`

Rescale the camera sensor dimensions and adjust focal length and principal point accordingly. Accepts either a new `(width, height)` pair or a scalar `scale` factor.

## Parameter string helpers

### `params_to_string()`

Return `params` as a comma-separated string, e.g. `"1200.0,960.0,540.0,0.01"`.

### `set_params_from_string(params_str)`

Parse a comma-separated string and overwrite `params`.

### `verify_params()`

Return `True` if the parameter vector has the correct length for the current model.

### `has_bogus_params(min_focal_length_ratio, max_focal_length_ratio, max_extra_param)`

Return `True` if any parameter is outside the valid range defined by the provided bounds.

## Camera models

All models report pixel coordinates. `cx` and `cy` are the principal point; `fx`, `fy` are focal lengths; `k1`, `k2`, `k3`, `k4` are radial distortion; `p1`, `p2` are tangential distortion.

| Model name | `params` layout |
|---|---|
| `SIMPLE_PINHOLE` | `f, cx, cy` |
| `PINHOLE` | `fx, fy, cx, cy` |
| `SIMPLE_RADIAL` | `f, cx, cy, k1` |
| `RADIAL` | `f, cx, cy, k1, k2` |
| `OPENCV` | `fx, fy, cx, cy, k1, k2, p1, p2` |
| `OPENCV_FISHEYE` | `fx, fy, cx, cy, k1, k2, k3, k4` |
| `FULL_OPENCV` | `fx, fy, cx, cy, k1, k2, p1, p2, k3, k4, k5, k6` |
| `FOV` | `fx, fy, cx, cy, omega` |
| `SIMPLE_RADIAL_FISHEYE` | `f, cx, cy, k1` |
| `RADIAL_FISHEYE` | `f, cx, cy, k1, k2` |
| `THIN_PRISM_FISHEYE` | `fx, fy, cx, cy, k1, k2, p1, p2, k3, k4, sx1, sy1` |

<Note>
  Use `camera.params_info` to confirm the exact layout for the currently set model.
</Note>

## Examples

```python
import pycolmap
import numpy as np

# Create via factory
cam = pycolmap.Camera.create(
    camera_id=1,
    model="OPENCV",
    focal_length=800.0,
    width=1280,
    height=720,
)
print(cam.params_info)  # fx, fy, cx, cy, k1, k2, p1, p2

# Project a normalized point to pixel coordinates
pix = cam.img_from_cam(np.array([0.1, -0.05]))

# Back-project a pixel to normalized camera coordinates
norm = cam.cam_from_img(np.array([680.0, 405.0]))

# Batch projection
points = np.random.randn(100, 2)
pixels = cam.img_from_cam(points)

# Verify parameters are valid
assert cam.verify_params()
```
