---
title: "essential_matrix_estimation"
description: "Estimate the essential matrix between two calibrated cameras from 2D point correspondences using LORANSAC and the 5-point algorithm."
---

## Function signature

```python
pycolmap.essential_matrix_estimation(
    points2D1,
    points2D2,
    camera1,
    camera2,
    estimation_options = pycolmap.RANSACOptions(),
)
```

<Note>
  Returns `None` if estimation fails. This happens when the RANSAC solver cannot find a model with enough inlier support. Always check the return value before accessing its fields.
</Note>

<Warning>
  The `estimation_options.max_error` is interpreted as a pixel threshold in image space. Internally it is converted to an angular error using the mean focal lengths of both cameras. The default is `4.0` pixels.
</Warning>

## Parameters

<ParamField path="points2D1" type="numpy.ndarray" required>
  Nx2 array of 2D pixel coordinates in image 1. Each row is `[x, y]`.
</ParamField>

<ParamField path="points2D2" type="numpy.ndarray" required>
  Nx2 array of 2D pixel coordinates in image 2. Each row is `[x, y]`. Must have the same number of rows as `points2D1`.
</ParamField>

<ParamField path="camera1" type="pycolmap.Camera" required>
  Camera model for image 1. Used to back-project image coordinates to normalized camera coordinates.
</ParamField>

<ParamField path="camera2" type="pycolmap.Camera" required>
  Camera model for image 2. Used to back-project image coordinates to normalized camera coordinates.
</ParamField>

<ParamField path="estimation_options" type="pycolmap.RANSACOptions">
  RANSAC configuration. See [RANSACOptions](#ransacoptions) below.
</ParamField>

## Return value

Returns a `dict` on success, or `None` on failure.

<ResponseField name="E" type="numpy.ndarray">
  3×3 essential matrix.
</ResponseField>

<ResponseField name="cam2_from_cam1" type="pycolmap.Rigid3d">
  Relative pose of camera 2 with respect to camera 1. The translation is recovered up to scale (unit norm).
</ResponseField>

<ResponseField name="num_inliers" type="int">
  Number of inlier correspondences after RANSAC.
</ResponseField>

<ResponseField name="inliers" type="list[bool]">
  Boolean mask of length N. `inliers[i]` is `True` when correspondence `i` is an inlier.
</ResponseField>

## RANSACOptions

Construct with `pycolmap.RANSACOptions()` and set fields, or pass a `dict`.

```python
options = pycolmap.RANSACOptions()
options.max_error = 2.0
options.confidence = 0.9999

# Equivalent dict form
pycolmap.essential_matrix_estimation(
    points2D1, points2D2, camera1, camera2,
    estimation_options={"max_error": 2.0, "confidence": 0.9999},
)
```

<ParamField path="max_error" type="float" default="4.0">
  Maximum reprojection error in pixels for an inlier correspondence.
</ParamField>

<ParamField path="min_inlier_ratio" type="float" default="0.01">
  Minimum fraction of correspondences that must be inliers for the model to be accepted.
</ParamField>

<ParamField path="confidence" type="float" default="0.9999">
  Desired probability of finding the correct model. Controls the number of RANSAC trials.
</ParamField>

<ParamField path="dyn_num_trials_multiplier" type="float">
  Multiplier on the dynamically computed number of trials.
</ParamField>

<ParamField path="min_num_trials" type="int" default="1000">
  Minimum number of RANSAC iterations regardless of the estimated number of trials.
</ParamField>

<ParamField path="max_num_trials" type="int" default="100000">
  Maximum number of RANSAC iterations.
</ParamField>

## Examples

<CodeGroup>

```python Basic usage
import numpy as np
import pycolmap

camera1 = pycolmap.Camera(
    model="SIMPLE_PINHOLE",
    width=1920,
    height=1080,
    params=[1200.0, 960.0, 540.0],
)
camera2 = pycolmap.Camera(
    model="SIMPLE_PINHOLE",
    width=1920,
    height=1080,
    params=[1200.0, 960.0, 540.0],
)

# Matched keypoint coordinates from two images
points2D1 = np.load("keypoints1.npy")  # (N, 2)
points2D2 = np.load("keypoints2.npy")  # (N, 2)

result = pycolmap.essential_matrix_estimation(points2D1, points2D2, camera1, camera2)

if result is not None:
    print("Essential matrix E:\n", result["E"])
    print("Relative pose:", result["cam2_from_cam1"])
    print("Inliers:", result["num_inliers"], "/", len(points2D1))
else:
    print("Estimation failed")
```

```python With custom RANSAC options
import numpy as np
import pycolmap

camera1 = pycolmap.Camera(
    model="OPENCV",
    width=1920,
    height=1080,
    params=[1200.0, 1200.0, 960.0, 540.0, -0.1, 0.05, 0.0, 0.0],
)
camera2 = pycolmap.Camera(
    model="OPENCV",
    width=1920,
    height=1080,
    params=[1180.0, 1180.0, 960.0, 540.0, -0.09, 0.04, 0.0, 0.0],
)

points2D1 = np.load("keypoints1.npy")
points2D2 = np.load("keypoints2.npy")

opts = pycolmap.RANSACOptions()
opts.max_error = 2.0
opts.min_num_trials = 2000
opts.max_num_trials = 200000

result = pycolmap.essential_matrix_estimation(
    points2D1, points2D2, camera1, camera2,
    estimation_options=opts,
)

if result is not None:
    inlier_pts1 = points2D1[result["inliers"]]
    inlier_pts2 = points2D2[result["inliers"]]
    print(f"{len(inlier_pts1)} inlier correspondences")
    # cam2_from_cam1.translation has unit norm (up to scale)
    print("Translation direction:", result["cam2_from_cam1"].translation)
```
