PyCOLMAP exposes the full COLMAP reconstruction pipeline as Python functions. Starting from a directory of overlapping images, you can produce a sparse 3D point cloud with calibrated camera poses, and optionally a dense point cloud via multi-view stereo.

The pipeline has four main stages:

1. **Feature extraction** — detect SIFT keypoints and compute descriptors for every image.
2. **Feature matching** — find corresponding keypoints across image pairs and verify them geometrically.
3. **Incremental mapping** — register images one at a time and triangulate 3D points (Structure-from-Motion).
4. **Dense reconstruction** — undistort images, estimate depth maps with patch-match stereo, then fuse them into a dense point cloud.

## End-to-end example

The following snippet runs sparse and dense reconstruction from a folder of images.

```python
import pathlib
import pycolmap

output_path = pathlib.Path("output/")
image_dir   = pathlib.Path("images/")

output_path.mkdir(exist_ok=True)
mvs_path      = output_path / "mvs"
database_path = output_path / "database.db"

# 1. Extract SIFT features
pycolmap.extract_features(database_path, image_dir)

# 2. Match features across all image pairs
pycolmap.match_exhaustive(database_path)

# 3. Incremental Structure-from-Motion
maps = pycolmap.incremental_mapping(database_path, image_dir, output_path)
maps[0].write(output_path)

# 4. Dense reconstruction (requires CUDA)
pycolmap.undistort_images(mvs_path, output_path, image_dir)
pycolmap.patch_match_stereo(mvs_path)          # requires CUDA
pycolmap.stereo_fusion(mvs_path / "dense.ply", mvs_path)
```

<Note>
  The PyPI wheels are not compiled with CUDA support. `patch_match_stereo` is only available when PyCOLMAP is built from source with CUDA enabled. See [Dense reconstruction](/pipeline/dense-reconstruction) for details.
</Note>

## Configuring options

Every pipeline function accepts option objects or plain Python dicts that are recursively merged into the built-in defaults. You only need to specify the keys you want to change:

```python
# Dict-based — only override what you need
pycolmap.extract_features(
    database_path,
    image_dir,
    sift_options={"max_num_features": 4096},
)

# Object-based — equivalent
opts = pycolmap.SiftExtractionOptions()
opts.max_num_features = 4096
pycolmap.extract_features(database_path, image_dir, sift_options=opts)
```

To inspect all available options and their default values:

```python
help(pycolmap.SiftExtractionOptions)
help(pycolmap.SiftMatchingOptions)
help(pycolmap.IncrementalPipelineOptions)
```
